# Catering Admin

A Personal Project for helping my mom to get esier for managing catering order.

## Getting Started

This project is serverless application for Admin Catering, I use Firebase as my server for saving data, such database.

Using:
  - BLoC
  - Equatable
  - Firebase Firestore
  - Firebase Storage
  - Firebase Crashlytic
  - Firebase Messaging
  - Shared Preference
  - Flutter Bloc
  - Google Map SDK (Soon)
